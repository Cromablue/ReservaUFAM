# Reservas UFAM
Sistema de reservas de salas de conferência, auditórios e veículos.
Autores: Evelim Bacury, Gabriel Moriz e José Santo.

🚧 EM DESENVOLVIMENTO 🚧


## Como acessar:
### Crie o banco de dados
```
python manage.py migrate
```

### Rode o servidor de desenvolvimento
```
python manage.py runserver
```

### Acesse no navegador
http://127.0.0.1:8000 (Clique com o Ctrl pressionado)
